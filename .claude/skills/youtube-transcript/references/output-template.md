# YouTube Transcript Output Template

## File Naming Convention

`YYYY-MM-DD-[video-title-slug].md`

Example: `2024-12-09-jordan-crawford-ai-gtm-interview.md`

## Folder Structure

- LinkedIn → `transcripts/youtube/linkedin/`
- Strategy → `transcripts/youtube/strategy/`
- Vibecode → `transcripts/youtube/vibecode/`
- Content → `transcripts/youtube/content/`

## Template

```markdown
# [Video Title]

**Source:** YouTube
**URL:** [YouTube URL]
**Channel:** [Channel Name]
**Published:** [Upload Date]
**Extracted:** [Today's Date]
**Use Case:** [LinkedIn | Strategy | Vibecode | Content]

---

## Video Description
[Video description if relevant]

---

## Transcript

[Formatted transcript content]

---

## Notes
[Optional: Any initial observations or context]
```
