# 25 TARGET ACCOUNTS FOR VAADIN

**ICP:** Mid-market to enterprise companies with Java-heavy tech stacks building internal tools, data-heavy dashboards, or maintaining legacy Java desktop applications (Swing/SWT/JavaFX). Industries: financial services, insurance, healthcare, logistics, industrial.

**Vaadin Sweet Spot:**
- 100-5,000 employees (Pro/Team/Enterprise tiers)
- Java backend teams building web UIs for internal or customer-facing apps
- Legacy Swing/SWT/JavaFX apps needing web modernization
- Regulated industries needing long-term support (15-year maintenance)
- Data-heavy applications: dashboards, ERPs, monitoring, trading

## VAADIN ICP SUMMARY

### Target Personas:
1. **CTO / VP Engineering** (Primary) - Owns modernization decisions and framework choices
2. **Engineering Manager** (Champion) - Evaluates tools, brings to leadership
3. **CIO** (Enterprise Budget) - Approves large modernization programs
4. **Java Developer / Lead** (End User) - Discovers Vaadin, advocates internally

### Company Characteristics:
- **Tech Stack:** Java backend (Spring Boot, Java EE, JSF)
- **Application Type:** Internal tools, dashboards, ERPs, trading platforms, clinical systems
- **Size:** 100-5,000 employees (Pro/Team tier), 5,000+ (Enterprise tier)
- **Geography:** US (37%), Germany (15%), France (9%) - primarily US
- **Pain Points:**
  - Legacy Swing/SWT/JavaFX apps expensive to maintain
  - Frontend/backend context switching slowing dev teams
  - Can't find Swing developers anymore (aging talent pool)
  - Need web/mobile accessibility for desktop-only apps
  - Oracle Java licensing cost increases
  - Compliance requirements need long-term framework support

### Key Buying Signals:
- Job postings mentioning Java Swing, legacy modernization, Vaadin
- PE acquisition or funding (= modernization budget)
- Cloud migration initiatives (moving desktop apps to web)
- Engineering blog posts about technical debt or modernization
- Hiring Java developers but struggling to find Swing experts
- Conference talks about their modernization journey
