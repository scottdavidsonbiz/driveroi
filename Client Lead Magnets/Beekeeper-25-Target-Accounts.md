# 25 TARGET ACCOUNTS FOR BEEKEEPER

**ICP:** Mid-large enterprises with 1,000+ frontline/deskless workers in hospitality, retail, manufacturing, construction, healthcare

---

## BEEKEEPER ICP SUMMARY

### What Beekeeper Does:
Frontline Success Platform - mobile-first app connecting deskless workers with communication, employee resources, work processes, and AI-powered tools all in one place.

### Target Personas:
1. **Chief People Officer / VP Human Resources** (Primary Economic Buyer)
   - Responsible for employee engagement, retention, and workforce communication
   - Owns budget for HR tech and employee experience tools
   - Pain: Frontline workers are disengaged, hard to reach, high turnover

2. **Internal Communications Manager** (Primary Champion)
   - Responsible for reaching all employees with company updates
   - Pain: 60%+ of workforce has no email, using insecure tools like WhatsApp
   - Success metric: Message reach rate, engagement, compliance

3. **VP Operations / COO** (Economic Buyer)
   - Responsible for operational efficiency, safety, compliance
   - Pain: Paper-based processes, inconsistent training, safety incidents
   - Success metric: Productivity, incident reduction, process digitization

4. **Chief Legal Officer / VP Compliance** (Influencer)
   - Responsible for labor law compliance, risk mitigation
   - Pain: Using unsecured messaging tools creates legal liability
   - Success metric: Compliance adherence, audit readiness

5. **CIO / VP IT** (Technical Buyer)
   - Responsible for technology stack, security, integrations
   - Pain: Fragmented tools, security risks, integration complexity
   - Success metric: System consolidation, security, adoption rates

### Company Characteristics:
- **Size:** 1,000+ employees with 60%+ frontline/deskless workers
- **Industries:** Hospitality, Retail, QSR, Manufacturing, Construction, Healthcare
- **Geography:** US (primary), Europe (secondary)
- **Locations:** Multi-location (10+ sites) or dispersed workforce
- **Tech maturity:** Still using SMS, WhatsApp, bulletin boards for communication

### Key Pain Points Beekeeper Solves:
1. **Communication gaps** - Frontline workers don't have email, miss critical updates
2. **Low engagement** - Disconnected from corporate, don't feel valued, high turnover
3. **Compliance risks** - Using unsecured messaging tools (WhatsApp, SMS)
4. **Operational inefficiency** - Paper checklists, manual processes, safety incidents
5. **Fragmented tech stack** - 5-10 different tools that don't integrate
6. **Language barriers** - Multi-lingual workforce, translation needs

### Buying Signals:
- Recent frontline hiring surge (opening 10+ new locations)
- High turnover rates (30%+ annual churn)
- Safety incidents or compliance violations
- Recent funding/acquisition (need to unify workforce)
- CFO asking for workforce productivity metrics
- Using WhatsApp/SMS for work communication
